# Dossier des Sprites

Ce dossier contient toutes les images 2D utilisées par le moteur graphique (Pygame) pour le jeu Pac-Man.

## Organisation :
- `/pacman/` : Sprites de base de Pac-Man (immobile, orienté).
- `/pacman/animations/` : Frames tirées de GIFs pour l'animation de mastication et de mort.
- `/ghosts/` : Sprites des fantômes (normal, vulnérable, clignotant). 
- `/maze/` : Sprites pour les éléments statiques (murs, pac-gommes).

**Rappel Important :** Utilisez toujours le format `.png` avec un fond transparent (Canal Alpha) pour que l’intégration visuelle sur Pygame soit réussie.
